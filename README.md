# CNN Object Detection using PyTorch

This repository demonstrates object detection using two popular Convolutional Neural Network (CNN) models implemented with PyTorch and TorchVision.

The notebooks run on **Google Colab** and perform object detection on uploaded images.

---

## Models Implemented

### 1. Mask R-CNN
Mask R-CNN performs **instance segmentation** along with object detection.

Features:
- Detects multiple objects in an image
- Generates **segmentation masks**
- Draws **bounding boxes**
- Displays **class labels and confidence scores**

### 2. Faster R-CNN

Faster R-CNN is a widely used CNN-based object detection model.

Features:
- Detects objects in images
- Draws **bounding boxes**
- Displays **class labels and confidence scores**
